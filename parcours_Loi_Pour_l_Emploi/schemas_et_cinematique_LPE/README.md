# 🔄 Cinématiques d’implémentation – Parcours Loi pour l’Emploi
 
Cet espace regroupe les **cinématiques d’implémentation** du parcours **Loi pour l’Emploi**.
 
Il met à disposition plusieurs **schémas Excalidraw complémentaires**, conçus pour faciliter la compréhension progressive du parcours, depuis une **vue d’ensemble** jusqu’au **détail API par API**.
 
---
 
## 🎯 Objectif de cet espace
 
Ces cinématiques ont pour objectif de :
 
- rendre lisible la **logique globale du parcours Loi pour l’Emploi**,
- visualiser l’**enchaînement des API**,
- faciliter l’appropriation par les équipes métier et techniques,
- servir de **référence commune** avant toute implémentation ou expérimentation.
 
---

## 🧭 Comment ouvrir les schémas Excalidraw
 
Les cinématiques sont fournies sous forme de fichiers **`.excalidraw`**.  
Ils peuvent être ouverts et consultés directement depuis l’outil en ligne Excalidraw.
 
### ▶️ Étapes pour ouvrir un schéma
 
1. Ouvrez le site : https://excalidraw.com/  
2. Une fois sur la page d’accueil, cliquez sur **☰ Menu** (en haut à gauche)
3. Sélectionnez **Open**
4. Chargez le fichier `.excalidraw` depuis votre poste
5. Le schéma s’ouvre alors dans l’éditeur Excalidraw
 
---
 
## 🔍 Bonnes pratiques de lecture
 
Pour une lecture confortable et efficace :
 
- utilisez le **zoom** pour naviguer entre vue d’ensemble et détails,
- déplacez-vous librement dans le schéma (clic + glisser),
- masquez ou affichez les éléments si nécessaire,
- commencez par la **vue d’ensemble**, puis explorez les schémas **API par API**.
 
👉 Excalidraw permet une lecture fluide, sans installation ni compte requis.
