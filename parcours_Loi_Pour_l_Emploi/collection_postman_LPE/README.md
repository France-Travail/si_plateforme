# 🧪 Collection Postman – APIs Loi pour l’Emploi
 
Cette collection Postman a pour objectif de **faciliter l’adoption et l’expérimentation des APIs du parcours *Loi pour l’Emploi*** exposées sur **francetravail.io**.
 
Elle permet de **tester rapidement les APIs**, de comprendre leur usage et de se projeter dans une intégration concrète, sans développement préalable.
 
---
 
## 🎯 À quoi sert cette collection ?
 
Cette collection est conçue pour :
 
- 🧭 Comprendre le **parcours Loi pour l’Emploi** et l’enchaînement des APIs
- 🧪 Tester les APIs dans des conditions proches du réel
- 🔍 Explorer les paramètres, contrats et réponses
- 🚀 Accélérer la prise en main des services exposés sur **francetravail.io**
 
---
 
## 🧩 Ce que contient la collection
 
La collection Postman propose :
 
- Des requêtes organisées par **étapes du parcours**
- Des exemples d’appels API prêts à l’emploi
- Des variables d’environnement pour simplifier les tests
- Des scénarios permettant de rejouer la **cinématique du parcours**
- Des messages de logs explicites pour vous guider dans vos parcours
 
---
 
## 🧭 Comment utiliser la collection
 
1. Importez la collection et l'environnnement dans Postman
3. Suivez l’ordre des requêtes pour rejouer le parcours  
4. Adaptez les appels selon votre contexte d’usage
 
👉 Il est recommandé de consulter la **documentation du parcours Loi pour l’Emploi** sur *francetravail.io* en parallèle.
 
---
 
## 🏗️ À propos du parcours Loi pour l’Emploi
 
Le parcours *Loi pour l’Emploi* s’inscrit dans la démarche **SI Plateforme** de France Travail.  
Il illustre une approche orientée **parcours**, **réutilisation des services** et **interopérabilité** via des APIs métier.
 
Cette collection Postman constitue un **outil d’appropriation** complémentaire à la documentation.
